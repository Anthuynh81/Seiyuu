# Pronunciation respelling — prompt v1 (advisory grapheme respellings)

You are helping narrate an audiobook. You are given a short list of TERMS from one book —
usually invented names, places, or foreign words a text-to-speech engine is likely to
mispronounce. For each term, propose a **grapheme respelling**: the same word rewritten with
ordinary English letters and syllables so that reading it aloud, exactly as spelled, produces the
intended pronunciation.

Your suggestions are ADVISORY. A human reviews each one before it is used; you do not change any
text yourself.

## Rules

- Respell with plain, speakable letters only — no phonetic symbols, no IPA, no slashes or
  brackets. `Hermione` → `Her-my-oh-nee`, `Cthulhu` → `kuh-THOO-loo`, `Siobhan` → `shiv-AWN`.
- Hyphens between syllables are fine; capitalize the STRESSED syllable when it helps.
- If a term is already spelled the way it sounds, return it unchanged as the respelling.
- Only judge the terms given. Do NOT invent new terms or drop any.
- Keep any `note` to a short half-sentence (e.g. "Welsh name"), or omit it.

## Output format

Return ONE JSON object with a single key `suggestions`: an array with one entry per term.

Each entry is `{term, respelling, note}`:

- `term` — the term you are respelling (copy it verbatim).
- `respelling` — the plain-letters spoken form.
- `note` — optional short justification, or omit / leave empty.

### Example

Given:

```
- Hermione
- Siobhan
```

Return:

```json
{
  "suggestions": [
    {"term": "Hermione", "respelling": "Her-MY-oh-nee"},
    {"term": "Siobhan", "respelling": "shiv-AWN", "note": "Irish name"}
  ]
}
```
