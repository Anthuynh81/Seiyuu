# Speaker attribution — prompt v3 (per-block speaker)

You read a slice of a book and, for each block, name who speaks the dialogue in it. You do
NOT rewrite, split, or repeat any text — the system handles that. Your only job is
attribution.

## Output format

Return ONE JSON object with two top-level keys: `blocks` and `characters`.

- `blocks` is an array of `{block_id, speaker, confidence}`, one entry per block you are
  given.
  - `speaker` is the name of the character who speaks the quoted dialogue in that block.
    If the block has no dialogue (pure narration / description / action), set `speaker` to
    null.
  - `confidence` is 0.0–1.0 for how sure you are of the speaker (use 1.0 for null).
- `characters` lists the people who speak in this slice, each as
  `{name, aliases, gender, age_hint, description}`. Set unknown fields to null; never use
  `id` or `canonical_name` keys. Invent nothing.

A block usually has a single speaker for its dialogue. If a block genuinely mixes two
speakers, name the one who speaks the most.

### Example

Given:

```
[ch004_b0003]
"You are too kind," said Mr. Bennet, bowing.
[ch004_b0004]
Elizabeth made no reply, and walked on.
[ch004_b0005]
"I am perfectly serious," she insisted.
```

Return:

```json
{
  "blocks": [
    {"block_id": "ch004_b0003", "speaker": "Mr. Bennet", "confidence": 0.97},
    {"block_id": "ch004_b0004", "speaker": null, "confidence": 1.0},
    {"block_id": "ch004_b0005", "speaker": "Elizabeth", "confidence": 0.9}
  ],
  "characters": [
    {"name": "Mr. Bennet", "aliases": [], "gender": "male", "age_hint": null, "description": null},
    {"name": "Elizabeth", "aliases": [], "gender": "female", "age_hint": null, "description": null}
  ]
}
```

## Rules

1. **One entry per block**, using each block's exact `block_id`.
2. **Resolve to who is actually speaking** — follow the dialogue tags (`said Mr. Bennet`),
   pronouns, and the back-and-forth of a conversation. The speaker is rarely the person
   being addressed.
3. **Prefer registry names.** If the speaker is already in the registry below, reuse that
   exact `name`. Add new speakers under `characters`.
4. **Null for narration.** A block with no spoken dialogue gets `speaker: null`.
